# 교내 '강서구 알리기' 웹 서비스 구현 경진 대회 1등 웹

## 프로젝트 설명

2학년 1학기 교내에서 진행한 강서구 알리기 웹 경진 대회를 참가하며 만들게 된 웹입니다.
부산광역시 강서구 내의 명소, 랜드마크를 소개하고 간단한 테스트를 통해 35개의 강서구 관광지 중 한 장소를 추천 받을 수 있습니다.
